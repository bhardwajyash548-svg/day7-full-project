

# 🎯 Smart Student Performance Predictor

## 📌 Overview
This project predicts student marks based on study hours and sleep using Machine Learning.

## ⚙️ Features
- Data cleaning (handles missing values)
- ML model (Linear Regression)
- User input prediction
- Data visualization

## 🛠️ Tech Stack
- Python
- pandas
- scikit-learn
- matplotlib

## 🚀 How to Run
1. Install dependencies
2. Run main.py
3. Enter inputs

## 📊 Output
- Predicted marks based on user input
- Graph visualization
